# My Portfolio – React + Tailwind

A modern, responsive developer portfolio built with React. Clean design, fast loading, and easy to maintain.  
Showcases projects, skills, and contact information with a focus on accessibility and performance.

---

## ✨ Features
- **Responsive UI** with mobile-first layout and sticky navbar (hamburger + animated dropdown).
- **Hero section** with intro, call-to-action buttons, and social links.
- **Projects grid** loaded from JSON, with images, tags, and links.
- **About section** with highlights, skills, and resume link.
- **Contact form** (EmailJS/Serverless optional).
- **Dark mode** support (system-preferred).
- **SEO basics**: meta tags, Open Graph, favicon.
- **Performance**: optimized images, lazy loading, code splitting.

---

## 🧰 Tech Stack
- **Frontend**: React + Tailwind CSS  
- **UI/Icons**: shadcn/ui, lucide-react, framer-motion  
- **Tooling**: ESLint, Prettier  
- **Deploy**: Vercel / Netlify / GitHub Pages
